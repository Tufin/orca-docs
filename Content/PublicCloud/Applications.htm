This screen lists your applications as specified by the tags associated with your assets. The following information is displayed for each application in the list:

  • # - sequence of item in the list
  • Application - Application name in the format specified in the App Manager
  • Assets - the number of assets associated with the application
  • Accounts - one or more accounts associated with the application's assets - cloud vendor icon () and account name defined in
  • Internet Access - 'Exposed' indicates one or more of the application's assets is exposed to the internet
  • Violations indicates one or more violations in one or more of the application's assets

  • Filter the application list - click Add a Filter
  • Display violations for an application - click violations icon
  • List assets for an application - click on an application in the list

Display Violations for an Application

If the violations icon is displayed for an application, you can click on it to display the application violations window. The issues will be listed, grouped by type with some types showing also the number of assets involved.
