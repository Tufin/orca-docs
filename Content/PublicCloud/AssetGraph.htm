This screen displays the connectivity of a single application, showing its associated nodes, the connections between them and connections with entities outside of the application. The arrows indicate that traffic is permitted, as defined by the security and network definitions such as VPC security objects and routing tables. To make viewing easier, icons can be moved around using the mouse.

Icons are displayed in red when there is a violation.

Reposition Icons

To make viewing easier, icons can be moved around the screen using the mouse. The default layout is restored whenever the screen is redisplayed.

View Node Details

To view details of an asset or other node associated with an application, click on the desired node. Properties are displayed according to the type of node:

  • For assets, asset details are displayed including network risky access, connectivity information, tags and virtual machine information. For more information, see Assets. If risky access exists, you can click on View effective access to see the details.

Drill Down to Assets

Double click on an asset to display the Asset Details or on a load balancer to display all assets related to it.
