Firewalls

locates and displays all and firewalls in your cloud accounts.

The screen displays a list of all firewalls found in your monitored accounts. It contains the following details:

  • Name - the name of the firewall
  • Account - the name of the account as configured in
  • Region - the geographical region of the firewall
  • Vnet - the network on which the firewall is deployed
  • Public IP - One or more IP addresses at which the firewall can be accessed from the internet
  • Private IP - The IP address of the firewall on the Vnet
  • View firewall details - click on the desired firewall from the list.

View Firewall Details

  1. Click on the desired firewall from the list. The Overview screen appears.

     

  2.  

    Switch between the Overview and Rules screens by clicking on Overview or Rules in the side menu.

Overview

 

The overview screen shows the tags assigned to the firewall device plus properties of the firewall that are not rule-specifc. When hovering over a property, the copy to clipboard icon appears - click to copy the value.

Rules

Azure Firewall Rules

The screen is split into NAT rules, network rules and application rules - click on the desired type of rule to switch between them.

The displayed rules are ordered by priority and the rule details displayed are as configured in your cloud account

  • Priority
  • Name
  • Source
  • Destination
  • Translated Destination (for NAT rules only)
  • Service
  • Translated Service (for NAT rules only)
  • Action (for network and application rules only)
  • Collection

AWS Firewall Rules

The screen is split into stateless and stateful rules. Switch between the two by clicking on the appropriate tab Stateless or Stateful. The AWS official documentation explains: "Network Firewall rule groups are either stateless or stateful. Stateless rule groups evaluate packets in isolation, while stateful rule groups evaluate them in the context of their traffic flow".

 

 

Rule details displayed are as configured in your cloud account.

  • Group (stateless only - the priority of the rule group)

  • Rule (stateless only)

  • Rule Group

  • Source

  • Destination

  • Service

  • Action

  • Direction (stateful only)

  • Rule Group Description
