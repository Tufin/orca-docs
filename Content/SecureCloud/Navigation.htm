Logging out

Click on your user icon at the top of the screen and then click on Log out from the window displayed.

Getting Help

The ? help icon is displayed at the top of every screen; click on it to display the Help Menu. Options:

  • Help center - takes you to the start page of the online help

  • API documentation - lists all the API calls available.
  • What can I do here? - takes you to the topic specific to the current screen
  • What's new - see the latest changes in
  • Take a tour - a brief look at the main functions you need to get started with
  • Contact us - links to the Tufin customer portal

Refreshing the Page

While you are working in the application, the data displayed on the screen could become outdated due to external events such as Kubernetes network policy changes or actions of other users such as adding allowed connections. When moving from one screen to another the new screen is always refreshed with the latest data but if you remain on the same screen, these changes are not automatically reflected.

The current page can however be refreshed manually by clicking on the Refresh icon at the top of the screen at any time.

Sorting Tables

Most tables can be sorted in ascending or descending order by one or more columns. Click on the column header of column to sort by the contents of that column. Click again to reverse the sort order.

Resizing Columns

Table column can be resized to make reading easier. Hover over a column divider in the header of a table, and drag the column separator bar right or left.

Filtering

The filtering options vary according to context:

While in screen, you can select cluster and namespaces. The current selection is displayed in the top bar and the data displayed is aggregated from all selected namespaces.

Filter by Namespace

Click on Namespace to change the selection. The filter icon is displayed when not all namespaces are selected. By default, displays a filtered view of the aggregated data in your cluster that includes all application namespaces and excludes system namespaces.

The currently selected filter remains in effect as you navigate to different screens

General Filtering

Some screens can be filtered by various criteria, relevant to the screen content.

  1. Click Add a filter

    The Add Filter window appears:

  2. Click Key and select the desired property from the list

  3. Click Operator and select the desired operator from the list. The available operators will vary according to context.

  4. Click Value and enter the value or select the value from the list

  5. Click Filter to proceed

  6. Repeat the steps above to add additional filters if required

  7. Edit a filter by clicking on it

  8. Remove a filter by clicking X on the filter

Filters can be saved by saving the full URL displayed when the filter is applied.

Filters are not retained when leaving the currently displayed screen.

Copy / Paste Fields on the Screen

A number of screens display the copy-to-clipboard icon when hovering over a field to allow easy copy/paste of screen information.

Download as CSV File

Some screens such as by Asset, and have the option of exporting and downloading the information displayed as a CSV file, which can be viewed using Excel and other CSV readers. When available, the CSV icon appears on the screen.
